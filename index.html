Current Research

We are currently working on:

  • Using Scala in Android applications.
  • Building a declarative, type-checked templating system that puts logic in ‘data definition’ templates, allowing multiple ‘view templates’ to both use the common logic code and to be be checked against the data definition templates to ensure that they are not, e.g., leaving out fields.
